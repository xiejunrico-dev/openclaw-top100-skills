Features

Everything you need,
nothing you don't

📐

Viewport Fitting

Every slide is exactly 100dvh — perfect on any screen, any device, any projector.

✏️

Inline Editing

Press E to edit directly in the browser. Ctrl+S saves to disk via File System Access API.

PPTX Export

Export to PowerPoint for stakeholders who need it. Your slides stay yours.
