05
Capabilities

Everything you need, nothing you don't

Purposefully minimal. Every feature serves the presentation — nothing serves the product team.

Viewport Fitting
Every slide fills exactly 100vh. Content that overflows gets split — not squished.
Inline Editing
Press E, click any text, edit directly. Ctrl+S saves without touching a code editor.
PPTX Export
/slide-creator --export pptx via Playwright and your local Chrome. Pixel-perfect output.
Two-Stage Workflow
--plan creates PLANNING.md outline first. --generate renders final HTML when ready.
06
Installation

Up in 30 seconds

Two platforms. One command each.

Claude Code
git clone https://github.com/kaisersong/slide-creator ~/.claude/skills/slide-creator
Restart Claude Code, then use /slide-creator
OpenClaw
clawhub install html-slide-creator
Auto-installs all dependencies. No additional configuration.
