Capabilities

Everything you need. Nothing you don't.

Purposefully minimal—every feature
serves the deck, not the product team.

VIEWPORT
Viewport Fitting
Every slide fills exactly 100vh. Content that overflows gets split — not squished.
EDIT
Inline Editing
Press E. Click text. Edit live. Ctrl+S saves. No code editor required.
EXPORT
PPTX Export
--export pptx via Playwright + your Chrome. Pixel-perfect PowerPoint output.
WORKFLOW
Two-Stage Flow
--plan creates PLANNING.md outline first. --generate renders HTML. Review before rendering.
