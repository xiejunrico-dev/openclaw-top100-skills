How It Works

Three steps to your
presentation

1

Run /slide-creator

In Claude Code or OpenClaw. State your topic and content goals.

2

Pick from 3 Visual Previews

Compare style options before committing. Choose by feel, not guesswork.

3

Open HTML

Self-contained file. No server. No install. Share as a file or host anywhere.

Two-stage workflow: Use --plan to outline first, then --generate to produce the full deck.

Features

Everything you need.
Nothing you don't.

📐

Viewport Fitting

100dvh slides. Pixel-perfect on any screen or projector.

✏️

Inline Editing

E key activates. Ctrl+S saves to disk via File System Access API.

📊

PPTX Export

Export for stakeholders who need PowerPoint. Note: backdrop-filter is flattened in PPTX.

⚡

Two-Stage Workflow

--plan to outline, --generate to produce. Ship faster, iterate smarter.
