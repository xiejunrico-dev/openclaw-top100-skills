Capabilities

Built for real use.

Viewport Fit Every slide fits exactly 100vh. No scrolling, no clipping — each layout is precision-designed for the viewport.
Inline Edit Press E, click any text, Ctrl+S to save. Edit mode built into every presentation. No export round-trip.
