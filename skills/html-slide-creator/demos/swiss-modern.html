06
Capabilities

Built for
real use.

Viewport Fit Every slide fits exactly 100vh. No scrolling, ever. Every layout is designed for the viewport.
Inline Edit Press E, click any text, Ctrl+S to save. Edit mode built into every presentation.
PPTX Export /slide-creator --export pptx converts to PowerPoint when the boardroom demands it.
Two-Stage --plan creates the outline. --generate produces final HTML. Review before committing.
