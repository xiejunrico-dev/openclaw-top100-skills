# 风控管理规则

> ⚠️ 量化交易风险控制指南

---

## 核心风控指标

### 1. 止损线（强制）

| 类型 | 阈值 | 动作 |
|:---|:---:|:---|
| **硬止损** | -8% | 立即执行止损，无需犹豫 |
| **软止损** | -5% | 评估是否提前止损 |
| **日内暴跌** | -5% | 紧急评估，考虑止损 |

**执行原则**:
- 触及-8%必须立即卖出
- 不得补仓摊低成本
- 不得心存侥幸等待反弹

### 2. 止盈线（建议）

| 类型 | 阈值 | 动作 |
|:---|:---:|:---|
| **第一止盈** | +5% | 减仓50% |
| **第二止盈** | +10% | 再减仓30% |
| **最终止盈** | +15% | 清仓或留底仓 |

**执行原则**:
- 达到+5%至少减仓一半
- 让利润奔跑，但要保住收益
- 回撤至+3%时清仓

### 3. 仓位管理

| 规则 | 限制 | 说明 |
|:---|:---:|:---|
| **单股最大仓位** | 30% | 防止单只股票风险过大 |
| **单板块最大仓位** | 50% | 分散板块风险 |
| **现金比例** | >10% | 保持流动性 |
| **杠杆使用** | 禁止 | 不使用融资融券 |

## 风险预警等级

### 🔴 红色预警（立即处理）
- 单股跌幅 ≥ -8%
- 单日总亏损 ≥ -5%
- 触及止损线

**动作**: 立即卖出，无需等待

### 🟡 黄色预警（密切关注）
- 单股跌幅 ≥ -5%
- 日内波动 ≥ ±5%
- 板块整体下跌 ≥ -3%

**动作**: 评估是否需要减仓，准备止损

### 🟢 绿色预警（正常监控）
- 单股跌幅 ≥ -3%
- 大盘下跌 ≥ -1%

**动作**: 正常监控，暂无需操作

## 特殊情形处理

### 1. 突发黑天鹅事件
- 立即评估持仓影响
- 如有重大利空，立即清仓
- 不赌反弹，先保命

### 2. 连续跌停
- 第一天跌停：评估是否卖出
- 第二天跌停：必须卖出（如有机会）
- 第三天跌停：接受损失，等待开板

### 3. 停牌风险
- 关注持仓股的公告
- 重大重组前减仓
- 避免重仓停牌股

### 4. 流动性风险
- 避免持仓市值过小的股票
- 日成交额低于1亿的股票谨慎买入
- 尾盘注意成交量萎缩

## 心理风控

### 禁止行为 ❌
- [ ] 冲动交易（未经计划）
- [ ] 报复性交易（亏损后加大赌注）
- [ ] 贪婪持有（达到止盈点不卖）
- [ ] 恐惧割肉（未达止损点提前卖）
- [ ] 过度自信（连续盈利后加大仓位）

### 建议行为 ✅
- [ ] 按计划交易
- [ ] 每日复盘得失
- [ ] 保持情绪稳定
- [ ] 接受亏损是交易的一部分
- [ ] 持续学习改进

- [ ] 确认风控线设置
- [ ] 评估隔夜市场风险

### 盘中每15分钟
- [ ] 检查持仓股涨跌幅
- [ ] 检查是否触发风控线
- [ ] 检查大盘走势

### 收盘后
- [ ] 统计当日最大回撤
- [ ] 评估当日操作质量
- [ ] 更新风控参数（如需要）
