## 🌟 About This Project

**OpenClaw Top 100** is a curated collection of the 100 most essential, high-quality skills for OpenClaw agents. Think of it like a premium warehouse store — fewer SKUs, but every single one is a best-in-class pick. Instead of drowning in 400+ options, you get exactly what you need to get started fast and get things done.

> **Philosophy:** 1–2 top skills per category. No duplicates, no noise. Just the essentials.

This list is **dynamic** — it is reviewed and updated monthly based on community usage, quality, and relevance. Skills that fall behind are replaced by better alternatives.

---

## 🚀 Quick Install

```bash
# Clone the repository
git clone https://github.com/LeoYeAI/openclaw-top100.git

# Copy a specific skill to your OpenClaw workspace
cp -r openclaw-top100/skills/<skill-name> ~/.openclaw/workspace/skills/

# Or install all 100 skills at once
cp -r openclaw-top100/skills/* ~/.openclaw/workspace/skills/
```

## 🔄 Dynamic Update Policy

This list is **not static**. Every month, we review the Top 100 against three criteria:

| Criterion | What We Check |
|---|---|
| **Community Usage** | Trending data from ClawHub and GitHub stars |
| **Quality Signals** | Description clarity, active maintenance, zero broken dependencies |
| **Coverage Gaps** | Every major use case must have at least one great skill |

Skills that fall behind are replaced by better alternatives. The goal is always: **the best 100, not the oldest 100**.
